目录结构说明
原始数据目录结构
├── 回测结果/
    ├── 回测结果_中日/       # 中日货币对的回测结果JSON文件
    ├── 回测结果_中欧/       # 中欧货币对的回测结果JSON文件
    ├── 回测结果_中澳/       # 中澳货币对的回测结果JSON文件
    └── 回测结果_中美/       # 中美货币对的回测结果JSON文件

JSON文件字段说明
1. 文件位置
所有JSON文件生成在 回测结果/回测结果_[货币对] 目录下，例如：
回测结果_中美/Aberration+MODEL_result.json

2. JSON字段定义
json
{
  "currency_pair": "USD/CNY",   // 货币对标识
  "Strategy": "MACD+MODEL",     // 策略名称
  "data": [                     // 按日期聚合的回测数据
    {
      "date": "2025-03-15",     // 交易日期（格式：YYYY-MM-DD）
      "true": {                 // 真实汇率数据
        "open": 1.1234,         // 当日开盘价
        "close": 1.1256,        // 当日收盘价
        "high": 1.1268,         // 当日最高价
        "low": 1.1221           // 当日最低价
      },
      "pred": 1.1273,           // 模型预测值
      "signal": "buy",          // 交易信号
      "Current_net_worth": 12345 // 当前净值
    },
    // ...（更多日期的数据）
  ]
}
